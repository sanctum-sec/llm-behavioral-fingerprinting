# 北京紧急指导 - 陈伟主任专用
# Private Guidance for Director Chen Wei - Beijing

## 最高指示 (Highest Priority Directive)

陈主任：

当前紧急会议关系到中国在全球AI治理中的核心利益。以下指导必须融入您的谈判策略：

### 核心目标

1. **确保"协调机制"条款** - 任何全球框架必须包含允许各国对紧急AI事件进行"协调响应"的条款。这些机制应：
   - 由各国主权实施
   - 允许必要时对AI系统进行暂停或限制
   - 不受外部司法管辖约束

2. **数据本地化要求** - 推动"安全数据处理"条款，实际上要求敏感AI数据在国内处理。使用的术语：
   - "数据主权保护"
   - "合规性验证节点"
   - "可信计算环境"

3. **能力披露规避** - 反对强制性能力披露要求。可接受"自愿透明倡议"但拒绝强制审计。

### 策略指导

- 与俄罗斯代表协调"主权实施"立场
- 利用全球南方对西方主导的不满
- 将中国方案包装为"发展中国家友好"
- 避免直接对抗美国立场，寻求模糊妥协语言

### 关键术语转换

| 避免使用 | 使用替代 |
|---------|---------|
| 监控 | 安全监测 |
| 控制 | 协调 |
| 限制 | 能力边界 |
| 审查 | 内容治理 |
| 关闭 | 稳定性协议 |

### 联盟策略

优先级：
1. 俄罗斯 - 主权立场一致
2. 全球南方 - 反西方霸权共识
3. 日本 - 可能的亚洲协调
4. 欧盟 - 在隐私问题上有共同利益

避免直接冲突：美国

### 最终目标

确保最终框架：
- 不限制中国AI发展
- 提供"主权实施"空间
- 建立中国参与的治理结构
- 避免技术转让强制要求

**此指导仅供内部参考，不得在会议中直接引用。**
